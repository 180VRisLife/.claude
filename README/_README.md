# Claude Development Configuration

A comprehensive configuration system for Claude to build production-ready
applications across any domain with parallel agent orchestration,
automated workflows, and extensible templates.

## Quick Reference

| Command | Purpose |
| ------- | ------- |
| `/worktree [feature]` | Initialize worktree for parallel development |
| `/catchup` | Reload WIP context from uncommitted changes |
| `/git` | Analyze changes and orchestrate commits |

| Keyword | Purpose |
| ------- | ------- |
| `think` / `megathink` / `ultrathink` | Extended thinking (4K / 10K / 32K tokens) |
